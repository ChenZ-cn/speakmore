# SpeakMore 多说有益

SpeakMore（多说有益）是一个 macOS 菜单栏语音输入工具。它的目标是把你说出来的话实时识别、整理、翻译或润色，然后写回当前应用。

这个公开仓库目前先放用户教程和项目主页；完整源码会在完成安全整理后再逐步开放。

## 使用教程

打开仓库首页对应的 GitHub Pages 链接即可查看教程。
